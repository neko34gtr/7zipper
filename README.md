# 7zipper (v1.0.0)

7zipperは、C# (.NET 9) で構築された、直感的な操作性を追求した7-Zip用フロントエンドツールです。ドラッグ＆ドロップだけで、煩わしい設定なしに「圧縮」と「解凍」を使い分けることができます。

## 🚀 主な特徴

- **インテリジェント・ドロップ**: ドロップ位置（左半分/右半分）に応じて、ファイルを「個別」にアーカイブするか「一括」でまとめるかを即座に選択。
- **自動解凍機能**: `.zip`, `.7z`, `.lzh` などのアーカイブをドロップすると自動的に解凍モードへ移行。
- **スマート・ディレクトリ展開**:
    - アーカイブ内にディレクトリ構造が含まれる場合：そのままの構造で展開。
    - アーカイブ内にディレクトリが含まれない場合：アーカイブ名でフォルダを作成して展開。
- **モダンな通知システム**: `MessageBox` を廃止。作業を妨げない **Windows標準トースト通知** と **タスクバーのフラッシュ** で完了をお知らせします。
- **設定の永続化**: `7zipper.json` (UTF-8 BOMなし) にウィンドウ位置や圧縮オプションを自動保存。

## 🛠 準備

本ツールは **7-Zip** の実行ファイルを外部から呼び出します。

1. [7-Zip](https://7-zip.org/) をインストールしてください。
2. デフォルトのパス（`C:\Program Files\7-Zip\7z.exe`）にインストールされていることを前提としています。

## 📖 使い方

1. `7zipper.exe` を実行します。
2. 対象のファイルやフォルダをメインウィンドウにドラッグ＆ドロップします。
    - **左側（「個別」）**: 各ファイルを個別に圧縮。
    - **右側（「一括」）**: 全てのファイルを一つのアーカイブに結合。
    - **アーカイブファイル**: 最適な構成で自動解凍。
3. ⚙（設定アイコン）をクリックすると、圧縮形式（.zip / .7z）や「最高圧縮」、「常に最前面」などの設定を変更できます。
